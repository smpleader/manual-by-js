Menu Deepth

There are 4 levels of content

1. An index file - Content section

It's html file where we use ManualByJs object

Example: feature.html to list down all pages for project features, while qa.html list down all page for quality assurance

We can put file link into Milestone Menu

2. Group of page

Use a node having only property "title" will start a new Goup in menu.json

It will generate a header for group of items in the menu sidebar

Example: "Getting Started", "Details", "Support" is a group in the current page

3. Page

It's an individual html file in your content folder

4. Page Index

It's a table index where we list down all the IDs of html document

This helps we indicate important heads in a page

Menu Types

1. Sidebar

List all the menu item and its group, show up page links in a Section of content.
All links are defined in menu.json inside content folder

2. Foot

3. Milestone

When develop a manual, we could have many version or sprint, this milestone menu show them so we recheck old content
